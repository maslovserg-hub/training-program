## Что это за проект
Личная годовая программа тренировок как одностраничный сайт (`index.html`).
Открыл на телефоне → видишь тренировку на сегодня → тренируешься. Живёт на GitHub Pages:
https://maslovserg-hub.github.io/training-program/ (репо maslovserg-hub/training-program, публичный).

## Текущее состояние
- Приложение собрано, задеплоено, **PWA** (ставится на Android). Логика «сегодня» по календарю, гири 2×16, дни Пн/Ср/Пт.
- **Дизайн**: тёмная + оранжевая тема, SVG-иконки, аккордеон на всю страницу (`name="acc"`), плавные анимации, фото-атлет (гиря в шапке/поле, бегун на день отдыха).
- **Программа пересмотрена** как тренером: см. `REVIEW.md`. День B — честная нагрузка низа (присед→пистолет) + задняя цепь (RDL/Nordic) + мобилка.
