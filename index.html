Тренировка

📅 Вся программа на год (52 недели)▾
текущая неделя разгрузка Год = 4 блока × 12 недель + разгрузка/тест
ℹ️ Как это работает▾
  • Прогрессия калистеники: делаешь вариант, который можешь чисто. Дошёл до верха повторов во всех подходах 2 тренировки подряд → следующий вариант.
  • Прогрессия гири: работаешь в диапазоне повторов. Добил верх во всех подходах → усложняй вариант (одноручный → двойной → темп). Тяжелее 16 кг не нужно.
  • Разгрузка: каждая 4-я неделя блока — 2 подхода, без отказа. Это часть прогресса, связки восстанавливаются.
  • Тест: недели 49–52 — лёгкая работа и проверка результата перед новым годовым кругом.
  • Каждая тренировка: разминка 8–10 мин → основная часть → заминка 5–10 мин.
📈 Усложнение упражнений▾

Когда текущий вариант стал лёгким — добил верх повторов во всех подходах 2 тренировки подряд — переходи к следующему. Это и есть прогрессия: не вес, а более сложный вариант движения.

Лесенка по твоей программе (по блокам года)
Общие прогрессии движений (от лёгкого к сложному)
Отжимания (горизонтальный жим)
  1. От стены
  2. С упором на возвышение (стол, скамья)
  3. С колен
  4. Классические
  5. Узкие / алмазные
  6. На кольцах / «лучник» (вес на одну руку)
  7. На одной руке
Подтягивания (вертикальная тяга)
  1. Вис + негативы (медленно вниз)
  2. Австралийские (горизонтальные, ноги на полу)
  3. Полные подтягивания
  4. С паузой вверху
  5. «Лучник» / разнохват
  6. На одной руке (сначала с резиной)
Приседания (ноги)
  1. Воздушные (без веса)
  2. Гоблет-присед с гирей
  3. Болгарские сплит-выпады
  4. Присед-пистолет с опорой
  5. Пистолет на одной ноге
Отжимания вниз (паралетсы / кольца)
  1. Отжимания на паралетсах
  2. Упор на кольцах + негативы (медленно вниз)
  3. Отжимания на кольцах в упоре (ring dips)
  4. Кольца с паузой / наклоном вперёд
  5. Взвешенные отжимания на кольцах (гиря в ногах)
Кор / пресс
  1. Планка
  2. Подъём согнутых ног лёжа
  3. Подъём прямых ног лёжа
  4. Подъём согнутых ног в висе
  5. Подъём прямых ног в висе
  6. L-sit / подъём ног к турнику
Баллистика гири
  1. Двуручный свинг
  2. Одноручный свинг
  3. Рывок (snatch)
  4. Толчок (clean & jerk)
  5. Длинный цикл (свинг → чистая → толчок)
Калистеника / скилы (Calimove-style)

Статические силовые элементы. Делай первым в тренировке, на свежие силы, короткими чистыми подходами. Прогресс месяцами — это норма.

Стойка на руках (handstand)
Видео техники
  1. Стойка у стены носом к стене
  2. Стойка у стены спиной + поиск баланса (отход от стены)
  3. Свободная стойка (выход в баланс)
  4. Отжимания в стойке у стены (HSPU)
  5. Свободные отжимания в стойке (free HSPU)
Планш (planche)
Видео техники
  1. Планш-наклон (planche lean) на паралетсах
  2. Подтянутый планш (tuck planche)
  3. Продвинутый tuck
  4. Straddle-планш (ноги в стороны)
  5. Полный планш (full planche)
Передний вис (front lever)
Видео техники
  1. Подтянутый (tuck), копи удержание
  2. Продвинутый tuck
  3. Одна нога / straddle
  4. Полный передний вис
  5. Подтягивания в переднем висе (продвинуто)
Задний вис (back lever)
Видео техники
  1. Подтянутый (tuck)
  2. Продвинутый tuck
  3. Одна нога / straddle
  4. Полный задний вис
Выход силой (muscle-up)
Видео техники
  1. Взрывные подтягивания (грудь к перекладине)
  2. Негативы выхода (медленно вниз из упора)
  3. Выход с резиной
  4. Чистый выход на турнике
  5. Выход на кольцах
L-sit → V-sit → Manna
Видео техники
  1. L-sit с опорой ног на паралетсах
  2. Подтянутый L-sit (tuck)
  3. Полный L-sit
  4. V-sit (ноги выше таза)
  5. Manna (продвинуто)
Флажок (human flag)
Видео техники
  1. Вертикальный флажок (тело вверх)
  2. Подтянутый (tuck)
  3. Straddle (ноги в стороны)
  4. Полный флажок

Опора — вертикальная стойка шведской стенки.
