# Как проверять «Тренировки»

## Как запустить
- Двойной клик по `index.html` — откроется в браузере. Сервер не нужен.
- На телефоне: открой файл в браузере → меню → «Добавить на главный экран». Дальше как приложение, офлайн.

## Что есть на странице (одна страница, разделы)
- **Шапка** — день тренировки (A/B/C), дата, бейджи: неделя N/52, блок, разгрузка/тест.
- **Блок-фокус** — что тренируем в текущем блоке.
- **Разминка** (сворачивается).
- **Основная часть** — 4 упражнения: название варианта, подходы×повторы, «Техника» (раскрывается).
- **Заминка** (сворачивается).
- **📅 Вся программа на год** — сетка 52 недель, тык по неделе → просмотр.
- **ℹ️ Как это работает** — правила прогрессии и разгрузки.
- **⚙️ Настройки** — дата старта, сдвиг ±1 тренировка, сброс сдвига.

## Чеклист проверки
- [ ] Открывается без интернета.
- [ ] В тренировочный день (Пн/Ср/Пт) показывает тренировку; день A/B/C чередуется.
- [ ] В день отдыха показывает «День отдыха» + дату следующей тренировки + кнопку «Посмотреть следующую».
- [ ] Бейдж «Неделя N / 52» меняется при смене даты старта.
- [ ] Каждая 4-я неделя блока (4, 8, 12 …) помечена «Разгрузка», подходы = 2.
- [ ] Недели 49–52 помечены «Тест».
- [ ] «Техника» у каждого упражнения раскрывается и сворачивается.
- [ ] В обзоре года тык по неделе открывает её; переключатель дней A/B/C работает; «← К сегодня» возвращает.
- [ ] Текущая неделя подсвечена в сетке.
- [ ] Кнопки сдвига ±1 двигают тренировку; «Сбросить сдвиг» возвращает.
- [ ] Дата старта сохраняется после перезагрузки (localStorage).

## Мобайл / адаптивность
- [ ] Нормально выглядит с 320px и шире.
- [ ] Кнопки крупные, попадаешь пальцем.
- [ ] Текст не вылезает, сетка недель переносится.

## Перед «релизом» (отдать себе на телефон)
- [ ] Пройдёт весь чеклист.
- [ ] Дата старта выставлена на реальный понедельник.
- [ ] Проверено в мобильном браузере, добавлено на главный экран.
