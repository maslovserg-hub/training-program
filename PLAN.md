# План: страница «Тренировка на сегодня»

## Что это
Одностраничное веб-приложение (`index.html`) поверх годовой программы тренировок.
Цель: открыл на телефоне — сразу видишь, что делать сегодня, и пошёл тренироваться.
Без раздумий, с учётом своего уровня.

## Кто и как пользуется
- Один пользователь (владелец), уровень — **средний**.
- Тренируется **3 раза в неделю: Пн / Ср / Пт**, по **45–60 мин**.
- Инвентарь: **2 гири по 16 кг**, турник, кольца, паралетсы, ролик для пресса, куб для прыжков.
- Открывает страницу в браузере телефона (офлайн, можно «добавить на главный экран»).

## Решения (зафиксированы на интервью)
| Вопрос | Решение |
|---|---|
| Формат | Один HTML-файл, без библиотек, офлайн |
| Что показывать | «Сегодня» на главном + отдельный обзор всего года |
| Логика «сегодня» | Авто по календарю (дата старта + дни Пн/Ср/Пт) |
| Страховка от пропусков | Кнопки сдвига ±1 день (не «галочки») |
| Функции | Показ тренировки дня + сворачиваемые подсказки по технике |
| Чего НЕ делаем | Галочек выполнения, записи весов/повторов, аккаунтов, сервера |
| Гиря | Две по 16 кг → прогрессия повторами и усложнением варианта, не весом |

## Как устроена программа (движок)
- Год = **4 блока по 12 недель** + 4 недели разгрузки/теста (49–52). После 52-й цикл повторяется (Год 2), стартуя со сложных вариантов.
  - Блок 1 (1–12) Фундамент · Блок 2 (13–24) Сила · Блок 3 (25–36) Навыки · Блок 4 (37–48) Пик.
- Неделя = **3 full-body тренировки A/B/C** с разным акцентом:
  - **A** — тяга-акцент (подтягивания, тяга, гиревой жим, кор)
  - **B** — ноги + баллистика (запрыгивания, свинги/рывки, присед/выпады, турецкий подъём)
  - **C** — жим-акцент (кольца, брусья/паралетсы, толчок, кор)
- У каждого упражнения **4 варианта сложности** (по одному на блок) + подсказка по технике.
- **Подходы/повторы** зависят от блока; каждая **4-я неделя блока — разгрузка** (2 подхода, без отказа).
- Текущая тренировка вычисляется: считаем число тренировочных дней от даты старта до сегодня → индекс сессии → неделя и день A/B/C.

## Хранение
- `localStorage`: дата старта программы + ручной сдвиг (offset). Больше ничего.

## Файлы проекта
- `index.html` — приложение (вся логика и данные внутри).
- `Программа_тренировок_год.md` — исходная методичка (источник содержания, остаётся как справочник).
- `CLAUDE.md` — правила работы над проектом.
- `TESTING.md` — как проверять.
- `PLAN.md` — этот файл.
