# Training Program

Личная годовая программа тренировок в виде одностраничного сайта.
Открыл на телефоне — видишь, что делать сегодня, и тренируешься. Без раздумий.

## Запуск

Открой `index.html` в браузере — больше ничего не нужно (работает офлайн).
На телефоне: открыть файл → меню браузера → «Добавить на главный экран».

## Что внутри

| Файл | Назначение |
|---|---|
| `index.html` | Приложение: вся логика, стили и данные в одном файле |
| `Программа_тренировок_год.md` | Исходная методичка (источник содержания) |
| `PLAN.md` | Что и зачем делаем |
| `TESTING.md` | Как проверять |
| `CLAUDE.md` | Правила работы над проектом |

## Коротко о программе

- Год = 4 блока по 12 недель + разгрузка/тест (49–52), дальше круг повторяется.
- 3 тренировки в неделю (Пн/Ср/Пт), full-body A/B/C, ~45–60 мин.
- Инвентарь: 2 гири по 16 кг, турник, кольца, паралетсы, ролик, куб.
- Прогрессия — повторами и усложнением варианта, не весом.

Стек: чистый HTML/CSS/JS, без библиотек и сервера. Состояние — `localStorage`.
